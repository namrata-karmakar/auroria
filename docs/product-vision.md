# Product Vision & Scope

## Goal:

Build an online jewelry shop where customers can browse products, add them to a cart, and place orders (MVP: no real payments required).

## Primary users:

1. Shoppers (browse, cart, checkout, view orders)

2. Admin / shop owner (manage products, see orders) – can come later.

## MVP success criteria (example):

A user can:

1. See a list of products.

2. See product details.

3. Add items to a cart.

4. “Checkout” and create an order.

5. Receive a basic order confirmation.
